# Full Stack Blog_Application 
## [Live Demo]([(https://blog-application-frontend-4b6k.onrender.com])
# Features
-Developed a full-stack MERN blog application with secure authentication using Firebase Google OAuth.
-Used React, Node.js, Express.js, and MongoDB technology to build this application.
-Deployed both frontend and backend on Render, ensuring environment-based configuration and production readiness.
-Implemented AI-powered blog description generation based on user-provided title and category to enhance content creation.
-Built RESTful APIs using Node.js and Express.js to support CRUD operations for blogs and user profiles.
-Integrated Multer and Cloudinary for efficient image upload and media storage.
-Implemented user features including blog creation, editing, deletion, likes, comments, and favorites.
-Added user profile management functionality, allowing users to update personal information.
